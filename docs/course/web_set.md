## 2. 勾选网页服务

![An image](./image/web_set/2.png)

## 3. 进行设置

图中外网IP 请填写自己服务器的外网IP

外网    例如：180.xxx.xxx.11
域名    例如：d.tapdrd.cn

注：非服务器/家用 无法直接通过外网IP访问

![An image](./image/web_set/3.png)

## 4. 本地测试

设置完毕后, 可以点击本地测试

将弹出到浏览器, 此时浏览器的网址应该是

127.0.0.1: 你设置的端口号, 80端口默认是不会显示端口号的

## 5. 外网测试

可以打开的你的手机浏览器

直接访问： IP: 端口, 即可访问

注：也可以用域名解析IP, 之后即可用域名访问
